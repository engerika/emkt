# emkt
assets


### 1. Direto (GitHub Raw)
> URL (Raw):
```https://raw.githubusercontent.com``` ```/engerika/emkt/``` ```**main** ou **hash**``` ```/nomearquivo```


### 2. Rápido (Via CDN)
> URL (jsDelivr):
``` https://cdn.jsdelivr.net/gh/engerika/emkto@hash/nomearquivo```


### 3. Performance (HTML)
> Tag de atributo loading="lazy": para imagens que não aparecem no topo da página.


### 4. Exemplos

*(Raw com main)*
```https://raw.githubusercontent.com/engerika/emkt/main/marca-simbolo_6.png```

*(CDN com main)*
```https://cdn.jsdelivr.net/gh/engerika/emkt@main/marca-simbolo_6.png```


*(Raw com hash)* ```https://raw.githubusercontent.com/engerika/emkt/51602987bec099a1117ee193b0580e565535ebe6/marca-simbolo_6.png```

*(CDN com hash)* ```https://cdn.jsdelivr.net/gh/engerika/emkt@51602987bec099a1117ee193b0580e565535ebe6/marca-simbolo_6.png```

*(HTML)* 
```<img 
  src="LINK" 
  alt="Nome" 
  width="800" 
  height="400" 
  loading="lazy"
```
